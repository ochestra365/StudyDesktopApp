## StudyDesktopApp에 대하여!
~~~
해당 챕터는 윈폼과 WPF를 이용한 것이다. object에 pix 태그를 달아서 알아보기 쉽게 표현해놓았지만
직관적이지 않다!
따라서 비주얼적으로 이를 관찰해야 학습에 도움이 될 것이다. 
내 생각엔 솔루션을 각각 켜서 해당 어플을 실행시켜보는 것이 복습에 도움이 될 거 같다!
~~~
